# Privacy Policy  
**BandiZZ Clicker**

Last updated: 01/12/2025

## 1. Data Collection  
The application **BandiZZ Clicker** collects only the following information:

### • User progression  
This includes the data required to save and restore the player’s progress in the game.  
No personal information is contained in these progression data.

### • Email address and password  
These are used solely to create and manage the user’s account.  
Passwords are securely stored (hashed) and are never visible or shared.

No other personal or technical data is collected.

## 2. Use of Data  
The collected information is used exclusively to:  
- allow users to create and access their accounts,  
- save and restore game progression.

The data is **not used** for advertising, analytics, tracking, or any other purpose.

## 3. Data Sharing  
**No data is shared** with third parties.  
No information is sold, transferred, or provided to external services.

## 4. Permissions  
The app uses only the permissions necessary for its basic functionality.  
It **does not access** any sensitive device data (photos, contacts, location, etc.).

## 5. Security  
We take measures to protect user information, including:  
- secure hashing of passwords,  
- no unencrypted transmission of data,  
- restricted access to account information.

## 6. Account Deletion  
Users may request the deletion of their account and all related data by contacting support.

## 7. Changes to This Policy  
This policy may be updated if the application evolves.  
The revision date will always be displayed at the top of this page.
